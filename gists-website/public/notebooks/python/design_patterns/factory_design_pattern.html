Factory Design Pattern¶

The Factory Design Pattern is a creational design pattern that provides an interface for creating objects in a superclass, but allows subclasses to alter the type of objects that will be created. It promotes loose coupling by eliminating the need to bind application-specific classes into the code. The pattern defines a method for creating the object, which subclasses can then implement to instantiate the desired object type.

Use Cases of Factory Design Pattern¶

  • When the exact type of object to create is determined at runtime.
  • When a system should be independent of how its objects are created and represented.
  • When creation logic is complex or involves multiple steps.
  • When code needs to follow the Open/Closed Principle for easy extension with new object types.
  • When multiple subtypes of an object need to be instantiated based on input, configuration, or context.

from abc import ABC, abstractmethod

# Step 1: Create a Product Interface
class Localizer(ABC):
    @abstractmethod
    def localize(self, message):
        pass

# Step 2: Concrete Product Classes
class EnglishLocalizer(Localizer):
    def localize(self, message):
        # English messages pass through unchanged
        return message

class FrenchLocalizer(Localizer):
    _translations = {
        "dog": "chien",
        "cat": "chat",
        "hello": "bonjour",
        "goodbye": "au revoir"
    }

    def localize(self, message):
        return self._translations.get(message, message)

class SpanishLocalizer(Localizer):
    _translations = {
        "dog": "perro",
        "cat": "gato",
        "hello": "hola",
        "goodbye": "adios"
    }

    def localize(self, message):
        return self._translations.get(message, message)

# Step 3: Factory Class
class LocalizerFactory:
    @staticmethod
    def create_localizer(language):
        if language == "english":
            return EnglishLocalizer()
        elif language == "french":
            return FrenchLocalizer()
        elif language == "spanish":
            return SpanishLocalizer()
        else:
            raise ValueError(f"Unknown language: {language}")

# Step 4: Client Code
if __name__ == "__main__":
    messages = ["dog", "cat", "hello", "goodbye", "unknown"]

    for lang in ["english", "french", "spanish"]:
        localizer = LocalizerFactory.create_localizer(lang)
        print(f"\nLocalizations in {lang.capitalize()}:")
        for msg in messages:
            print(f"  {msg} -> {localizer.localize(msg)}")
